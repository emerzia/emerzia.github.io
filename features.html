FEATURES

List of features offered by Emerzia, Emergency Notification System.

What is Reverse Emergency Dialering?

Reverse emergency dialling using an auto-dialer, automatically calls thousands of numbers from a phone list or database.

Once the call has been answered, the autodialer either plays a recorded message or connects the call to a live person. Emerzia has a "Live Person Confirmation" feature allowing the contact to indicate that they have heard and understood the message by pressing a button on their handset. This data recorded in the comprehensive reporting system.

Features


FLEXIBLE IVR SYSTEM

Complex IVR trees can be designed to collect information from the contact and includes recording, multi-choice, extension dialling, and conferencing nodes.

 

HIGH CAPACITY

Tests on a mid-range server resulted in 1 million calls in just under 5 hours. Multi-Server systems will result in much higher capacity.

 

VOICEMAIL DETECTION

Voicemail Detection enhances the system by allowing to detect the call has been answered by voicemail, and leave a different message, or simply hangup.

 

REALTIME REPORTING

Capacity monitoring and other key performance indicators are reported in realtime, with downloadable reports and drill-down to individual survey results.

 

SCALABLE / HIGH AVAILABILITY

Multi-server systems can be clustered to create a highly fault tolerant and scalable system.

 

ADAPTIVE WEB PAGES

Web management interfaces adapt to the size of browser being used, making Emerzia controllable by smart phone, tablet, laptop and desktop devices.

 

USER FRIENDLY

Intuitive web interface based customer and management interfaces requiring minimal operator training.

 

API INTEGRATION

API's have been written to allow straightforward integration with third party applications.

 

SMS BROADCASTING

SMS integration into Emerzia allows mass broadcasting of text messages to cell-phones.

 

CONTACT ACQUISITION

A sign up page is provided to allow the community to register their phone details online with Emerzia and select which sort of alerts they want, for instance school closures through to distaster and evacuation alerts such as bush-fires.
